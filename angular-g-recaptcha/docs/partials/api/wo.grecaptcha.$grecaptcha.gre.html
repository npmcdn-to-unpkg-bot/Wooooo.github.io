gre
object in module wo.grecaptcha

Description

Grecaptcha instance that contains parameters and element, wigetId. Returned by $grecaptcha service.

Methods

  • get(key)

    Get the value of gre config's 'key' property

    Parameters
    ParamTypeDetails
    keystring

    the key of parameters to get

    Returns
    *

    a value of the key

    Example

    var gre = $grecaptcha({theme: 'dark'});

gre.get('theme'); // 'dark'

gre.get('type'); // 'image' ('image' is defualt type.)

  • getElement()

    Get the element where recaptcha box is rendered if exists.

    Returns
    object

    jQuery object wrapping element where recaptcha box is rendered

    Example

    var gre = $grecaptcha();

var el = $document[0].createElement('div');


gre.render(el).then(function(){
     gre.getElement(); // jQuery object wrapping 'el'
};

  • getResponse()

    Get the gre's response from recaptcha box if exists.

    Returns
    string

    response

    Example

    var gre = $grecaptcha();

// after validating phase
gre.getResponse(); // will give response of recaptcha box

  • getWidgetId()

    Get the recaptcha box's widget id if exists.

    Returns
    number

    widget id

    Example

    var gre = $grecaptcha();
var el = $document[0].createElement('div');

gre.render(el).then(function(){
     gre.getWidgetId(); // 0
});

  • init(onInit)

    Set private grecaptcha object from $window after recaptcha script loaded if undefined.

    By returning a promise which be resolved on script loaded,
    can check whether grecaptcha initialization is over or not.

    Parameters
    ParamTypeDetails
    onInit
    (optional)
    		Function

    a callback performed after initialization is finished

    Returns
    Promise

    Promise that will be resolved when onload callback be execute.
    It will be rejected if times out.

    Example

    var gre = $grecaptcha();

gre.init(function(){
     console.log('Initializing!!');
}).then(function(){
     console.log('Init is over.');
});

  • remove()

    Delete the gre instance from private map object and greList.

    Example

    var gre = $grecaptcha();
var el = $document[0].createElement('div');

gre.render(el).then(function(){
     el.remove();
});

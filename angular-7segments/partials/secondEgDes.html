• A dot resides in a digit with other character together because angular-7segments reflects real seven segments.
  • However, if dot's left is either a dot or a space, then the dot occupies isolated digit.
    eg) 1..1 .. -> [1.] [.] [1] [.] [.] ( [ ] means one digit. )
  • Also, if a dot is the first character, then the dot holds one distinct digit.
    eg) .1. -> [.] [1.]
  • Unless, a dot resides in one digit with other friend.
    eg) 1.1.1. -> [1.] [1.] [1.]
